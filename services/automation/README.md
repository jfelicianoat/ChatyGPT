# Servicio local Python

Reservado para automatizaciones, integración avanzada con documentos y tareas
programadas de las fases 2–4.

No se inicia ningún proceso Python en la Fase 0: la migración, la recuperación y
la conexión con Broker AI pertenecen al núcleo Rust. Antes de añadir este
servicio se definirá un protocolo local autenticado, versionado y con ciclo de
vida controlado por Tauri.
